## BRC20-S

- 基于OKX提出的BRC20-S标准,设计目的是使开发者和用户更容易使用。用户可以通过质押BRC20代币赚取收益,BRC20-S代币是BRC20代币功能上的扩展,同时添加了关于质押协议的描述。大大的推动了BRC20生态系统的发展。
